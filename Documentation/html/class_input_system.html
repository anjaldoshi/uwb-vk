Detailed Description

Input handling using GLFW.

This class handles all of the inputs recieved from windows through GLFW. Down states, pressed states, and release states for both mouse and keyboard can be queried. Note: in addition to the standard GLFW callbacks, this class uses dispatch methods as GLFW can only use static methods for the callback.
