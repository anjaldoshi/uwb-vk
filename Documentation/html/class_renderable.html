Detailed Description

A Class for representing objects that are rendered in the scene.

This class Represents objects that are rendered within a scene in the RenderSystem. It consists of: a mesh, a shaderSet, Bindings, A descriptorSetLayout, descriptorSets, a pipeline layout, and a pipeline.

VkShaderStageFlagBits  stage,
uint32_t  bindingNum,
uint32_t  count 
)

Note a specific binding expected by the shaders.

This method does not actually bind resources but prepares a list of what bindings to expect. When actually binding resources, this list is checked to ensure that all the expected resources are filled and there are no duplicates, etc.

Parameters
descriptorTypeThe type of descriptor to use (i.e image/sampler or uniform buffer)
stageThe stage the binding is at
bindingNumThe number to bind at
countThe number of resources of that type to bind to the given bindingNum

Create and write the VkDescriptorSets that will by used by this Renderable.
